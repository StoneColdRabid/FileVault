📖 FileVault Help

Getting Started

  • Click Open Directory (top-right) to select any folder or drive.
  • Your browser will ask for permission — click Allow or View Files.
  • FileVault scans the directory and lists all files organized by type.
  • Set Scan depth before opening — higher values scan more subdirectories (may be slow for large drives).

Browsing & Filtering

  • Use the sidebar on the left to filter files by type (Images, Video, Code…).
  • Use the search bar to find files by name.
  • Click column headers to sort by name, type, size, or date.
  • Toggle List / Grid view with the toolbar buttons.

Selecting Files

  • Click a row's checkbox to select individual files.
  • Use the header checkbox or Select All to select all visible files.
  • The Action Bar appears at the bottom when files are selected.
  • ⌘A — select all   Esc — clear selection / close modal

File Operations

  • Copy To… — Copy selected files to a folder you choose.
  • Move To… — Move selected files (copy then delete originals).
  • Rename — Rename a single selected file (one at a time).
  • Open — Open file(s) in a new browser tab (great for images, PDFs, text).
  • Delete — Moves selected files to macOS Trash via server.py. Requires the root path field in the header to be set correctly.

Browser Requirements

  • Requires the File System Access API.
  • Fully supported in Google Chrome, Microsoft Edge, and Brave.
  • Safari on macOS Tahoe has partial support — Chrome is recommended.
  • Firefox does not currently support this API.

Privacy

FileVault is 100% local. No data, files, or metadata leave your machine. Everything runs in your browser with no network calls.
